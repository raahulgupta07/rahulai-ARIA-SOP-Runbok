Desktop Sync — setup

Best when you can't get Microsoft admin help. The desktop’s OneDrive client does the sign-in (already approved by your org) — Aria never touches Microsoft.
  1. On the SharePoint library in your browser → click Sync (opens OneDrive, mirrors it to a local folder).
  2. Install the Aria desktop agent on that machine (one binary):
